## Overview
A modern Next.js application built with Shadcn UI and Tailwind CSS for the ServiceHub platform. Features a premium landing page, basic authentication flows, and dashboard layouts.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000) with your browser.
